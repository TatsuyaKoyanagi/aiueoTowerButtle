# あいうえおタワーバトル！ 

## 概要 (Overview)
大小様々なひらがなの「あいうえお」を、崩れないように高く積み上げていく物理演算パズルゲームです。
初めてのゲーム制作プロジェクトとして、Unityの基本操作と物理エンジンの挙動を学ぶために開発しました。

> **Note**
> 現在開発中のため、Unityエディタ上での動作を想定しています。

## ゲームプレイ (Gameplay)
ランダムに出現する文字をバランスよく積み重ね、タワーの高さを競います。文字の形状（「あ」や「い」など）によって重心や引っかかりやすさが異なるため、配置の戦略性が求められます。

## 操作方法 (Controls)
マウスのみで操作可能です。

| 操作 | アクション | 詳細 |
| :--- | :--- | :--- |
| **左クリック (長押し)** | **移動** | マウスカーソルに合わせて文字が左右に移動します。 |
| **左クリック (離す)** | **落下** | 文字をその場から落下させます。 |
| **右クリック (長押し)** | **回転** | 落下前に文字の角度を調整します。 |

## 使用技術 (Tech Stack)
* **Engine**: Unity 2022.3.0f1
* **Language**: C#
* **Features**:
    * Rigidbody / Colliders (物理演算)

## 開発の背景・工夫点 (Background)
**初めてのゲーム開発**への挑戦として作成しました。
* **物理挙動の調整**: 文字ごとの形状に合わせてコライダーを設定し、積み上げた際の予想外の動き（崩落）を楽しめるように調整しました。
* **操作性の追求**: 直感的に遊べるよう、マウス操作とオブジェクトの追従性の同期に注力しました。
